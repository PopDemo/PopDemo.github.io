  • lombok实现原理和背景
    • 背景
      • 自从java6开始，javac就开始支持“JSR 269 Pluggable Annotation Processing Api”规范，也就是只要程序实现该API，就能在javac运行时得到调用
    • LomBok就是一个实现了“JSR 269 API”的程序，在使用javac的过程中，它产生作用的具体流程
      • javac对其源码进行分析，生成抽象语法树（SAT）
